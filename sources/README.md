# Source Access Notes

The repository does not redistribute full third-party documents. Direct links, page or section locators, publication dates, and intended uses are recorded in [`../data/source_catalog.csv`](../data/source_catalog.csv).

## Source policy

- Prefer official BSI and OJK disclosures for incident facts.
- Use the audited BSI financial statements for the year-end management disclosure.
- Use Reuters only to document the existence of an external LockBit claim and BSI's reported response.
- Use OJK rules, Indonesian law, and NIST publications for control expectations and recommendations.
- Do not use screenshots from dark-web sites, leaked datasets, social-media rumors, or alleged stolen records.

## Access limitation

Some BSI Investor Relations PDF endpoints use automated-access protection. Open those links in a normal browser if a command-line download is blocked. A blocked automated download does not change the source classification or justify replacing the source with an unverified copy.

## Reproducibility

The analysis can be reproduced from the CSV tables without downloading the source PDFs. A reviewer should use the source catalog to verify important claims against the publisher's current public copy.
